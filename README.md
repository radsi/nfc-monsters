# NFC Monsters 👺 / WIP

NFC Monsters is a mobile roguelike built in Godot, inspired by Slay the Spire, where combat actions, spells, and upgrades are controlled through NFC tags.

Instead of selecting cards on screen, players use real NFC cards that represent:

- Attacks
- Upgrades
- Discounts
- Other effects

This is a concept I've always thought about, so I decided to make a game to see how it would work.
